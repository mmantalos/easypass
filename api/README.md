# Back-end

## Περιεχόμενα:
- RESTful API

## Εγκατάσταση
Συμπληρώστε τα στοιχεία της βάσης στο config.json
```sh
npm install
./create_credentials.sh
```

## Εκκίνηση
```sh
node index.js
```

## Endpoints
Τα Endpoints που υποστηρίζονται από το RESTful API μας βρίσκονται στο directory: api/endpoints
