# test-backend

## Περιεχόμενα:
- Backend tests

## Προαπαιτούμενα:
- Apache JMeter
- Java Runtime

## Χρήση:
- Ανοίξτε το Apache JMeter
- Αλλάξτε το θέμα σε System Default (σε περίπτωση που δεν λειτουργεί το "Open file")
- Κάντε open το αρχείο 'Spike Test.jmx' ή 'Stress Test.jmx'
- Τρέξτε το test
- Επιλέξτε από το αριστερό μενού την μορφή στην οποία θέλετε να δείτε τα αποτελέσματα
